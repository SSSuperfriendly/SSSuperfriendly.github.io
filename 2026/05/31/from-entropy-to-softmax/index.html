连接主义的包容：从 Logistic 到 Softmax 回归

对于一个离散随机变量 XXX，其概率分布为 PPP，香农熵定义为：

H(P)=−∑kp(k)log⁡p(k) H(P) = -\sum_k p(k) \log p(k) H(P)=−k∑​p(k)logp(k)

熵是混乱程度的度量，熵上升意味着纯度下降。

考虑一个真实分布为 PPP，我们用分布 QQQ 来编码事件，定义交叉熵为：

H(P,Q)=−∑kp(k)log⁡q(k) H(P, Q) = -\sum_k p(k) \log q(k) H(P,Q)=−k∑​p(k)logq(k)

如何度量 QQQ 和 PPP 的相似程度，也就是如何根据训练样本选择最好的模型参数？这里我们引入最大似然估计。

假设训练集为 D={(x(i),y(i))}i=1N\mathcal{D} = \{(\boldsymbol{x}^{(i)}, y^{(i)})\}_{i=1}^ND={(x(i),y(i))}i=1N​，包含 NNN 个独立同分布样本。我们希望模型在给定输入 x\boldsymbol{x}x 时输出对标签 yyy 的预测概率 pθ(y∣x)p_{\boldsymbol{\theta}}(y \mid \boldsymbol{x})pθ​(y∣x)，其中 θ\boldsymbol{\theta}θ 代表如权重 w\boldsymbol{w}w 和偏置 bbb 一样的参数。

一个直接的标准是选择参数 θ\boldsymbol{\theta}θ，使模型观测到这些真实标签的概率最大。这个概率就是似然函数：

L(θ)=∏i=1Npθ(y(i)∣x(i)) L(\boldsymbol{\theta}) = \prod_{i=1}^N p_{\boldsymbol{\theta}}\bigl(y^{(i)} \mid \boldsymbol{x}^{(i)}\bigr) L(θ)=i=1∏N​pθ​(y(i)∣x(i))

对其取负对数，目标就变成最小化负对数似然：

L(θ)=−ℓ(θ)=−∑i=1Nlog⁡pθ(y(i)∣x(i)) \mathcal{L}(\boldsymbol{\theta}) = -\ell(\boldsymbol{\theta}) = -\sum_{i=1}^N \log p_{\boldsymbol{\theta}}\bigl(y^{(i)} \mid \boldsymbol{x}^{(i)}\bigr) L(θ)=−ℓ(θ)=−i=1∑N​logpθ​(y(i)∣x(i))

训练目标就是最小化 L(θ)\mathcal{L}(\boldsymbol{\theta})L(θ)。

对于分类标签，真实分布可以看作一个极端概率分布，也就是 one-hot 向量，真实类概率为 111，其余为 000。于是有：

−log⁡pθ(y(i)∣x(i))=−∑k1{y(i)=k}log⁡pθ(k∣x(i))=H(Pi,Qi) -\log p_{\boldsymbol{\theta}}\bigl(y^{(i)} \mid \boldsymbol{x}^{(i)}\bigr) = -\sum_k \mathbf{1}\{y^{(i)} = k\} \log p_{\boldsymbol{\theta}}\bigl(k \mid \boldsymbol{x}^{(i)}\bigr) = H(P_i, Q_i) −logpθ​(y(i)∣x(i))=−k∑​1{y(i)=k}logpθ​(k∣x(i))=H(Pi​,Qi​)

这正是每个样本的真实分布 PiP_iPi​ 与模型预测分布 QiQ_iQi​ 的交叉熵。所以，最大化似然等价于最小化交叉熵损失。

Logistic 回归

对于二分类，标签 y∈{0,1}y \in \{0, 1\}y∈{0,1}。我们假设它服从伯努利分布，其唯一参数为 P(y=1)P(y = 1)P(y=1)。Logistic 回归让线性组合经过 sigmoid 函数来输出这个概率：

z=w⊤x+b,y^=σ(z)=11+e−z z = \boldsymbol{w}^\top \boldsymbol{x} + b, \qquad \hat{y} = \sigma(z) = \frac{1}{1 + e^{-z}} z=w⊤x+b,y^​=σ(z)=1+e−z1​

于是：

P(y=1∣x)=y^,P(y=0∣x)=1−y^ P(y = 1 \mid \boldsymbol{x}) = \hat{y}, \qquad P(y = 0 \mid \boldsymbol{x}) = 1 - \hat{y} P(y=1∣x)=y^​,P(y=0∣x)=1−y^​

统一写为：

p(y∣x)=y^y(1−y^)1−y p(y \mid \boldsymbol{x}) = \hat{y}^y (1 - \hat{y})^{1 - y} p(y∣x)=y^​y(1−y^​)1−y

整个训练集的似然函数为：

L=∏i=1Ny^iyi(1−y^i)1−yi L = \prod_{i=1}^N \hat{y}_i^{y_i} (1 - \hat{y}_i)^{1 - y_i} L=i=1∏N​y^​iyi​​(1−y^​i​)1−yi​

取对数，得到对数似然：

ℓ=∑i=1N[yilog⁡y^i+(1−yi)log⁡(1−y^i)] \ell = \sum_{i=1}^N \bigl[y_i \log \hat{y}_i + (1 - y_i)\log(1 - \hat{y}_i)\bigr] ℓ=i=1∑N​[yi​logy^​i​+(1−yi​)log(1−y^​i​)]

最大化 ℓ\ellℓ 等价于最小化其负值，也就是二元交叉熵损失：

L=−∑i=1N[yilog⁡y^i+(1−yi)log⁡(1−y^i)] \mathcal{L} = -\sum_{i=1}^N \bigl[y_i \log \hat{y}_i + (1 - y_i)\log(1 - \hat{y}_i)\bigr] L=−i=1∑N​[yi​logy^​i​+(1−yi​)log(1−y^​i​)]

为什么它能避免梯度消失？

先看单样本损失：

L=−[ylog⁡σ(z)+(1−y)log⁡(1−σ(z))] \mathcal{L} = -\bigl[y\log\sigma(z) + (1 - y)\log(1 - \sigma(z))\bigr] L=−[ylogσ(z)+(1−y)log(1−σ(z))]

求 ∂L∂w\frac{\partial \mathcal{L}}{\partial \boldsymbol{w}}∂w∂L​。由链式法则，

∂L∂w=∂L∂z⋅∂z∂w=∂L∂z⋅x \frac{\partial \mathcal{L}}{\partial \boldsymbol{w}} = \frac{\partial \mathcal{L}}{\partial z} \cdot \frac{\partial z}{\partial \boldsymbol{w}} = \frac{\partial \mathcal{L}}{\partial z} \cdot \boldsymbol{x} ∂w∂L​=∂z∂L​⋅∂w∂z​=∂z∂L​⋅x

先求 ∂L∂z\frac{\partial \mathcal{L}}{\partial z}∂z∂L​：

∂L∂z=−y⋅1σ(z)σ′(z)−(1−y)⋅11−σ(z)(−σ′(z))=−yσ′(z)σ(z)+(1−y)σ′(z)1−σ(z) \begin{aligned} \frac{\partial \mathcal{L}}{\partial z} &= -y \cdot \frac{1}{\sigma(z)}\sigma'(z) - (1 - y) \cdot \frac{1}{1 - \sigma(z)}\bigl(-\sigma'(z)\bigr) \\ &= -y \frac{\sigma'(z)}{\sigma(z)} + (1 - y)\frac{\sigma'(z)}{1 - \sigma(z)} \end{aligned} ∂z∂L​​=−y⋅σ(z)1​σ′(z)−(1−y)⋅1−σ(z)1​(−σ′(z))=−yσ(z)σ′(z)​+(1−y)1−σ(z)σ′(z)​​

代入 sigmoid 的导数 σ′(z)=σ(z)(1−σ(z))\sigma'(z) = \sigma(z)\bigl(1 - \sigma(z)\bigr)σ′(z)=σ(z)(1−σ(z))：

∂L∂z=−yσ(z)(1−σ(z))σ(z)+(1−y)σ(z)(1−σ(z))1−σ(z)=−y(1−σ(z))+(1−y)σ(z)=−y+yσ(z)+σ(z)−yσ(z)=σ(z)−y \begin{aligned} \frac{\partial \mathcal{L}}{\partial z} &= -y \frac{\sigma(z)\bigl(1 - \sigma(z)\bigr)}{\sigma(z)} + (1 - y)\frac{\sigma(z)\bigl(1 - \sigma(z)\bigr)}{1 - \sigma(z)} \\ &= -y\bigl(1 - \sigma(z)\bigr) + (1 - y)\sigma(z) \\ &= -y + y\sigma(z) + \sigma(z) - y\sigma(z) \\ &= \sigma(z) - y \end{aligned} ∂z∂L​​=−yσ(z)σ(z)(1−σ(z))​+(1−y)1−σ(z)σ(z)(1−σ(z))​=−y(1−σ(z))+(1−y)σ(z)=−y+yσ(z)+σ(z)−yσ(z)=σ(z)−y​

因此：

∂L∂w=(σ(z)−y)x \frac{\partial \mathcal{L}}{\partial \boldsymbol{w}} = \bigl(\sigma(z) - y\bigr)\boldsymbol{x} ∂w∂L​=(σ(z)−y)x

对于 NNN 个样本，梯度就是求和或平均：

∂L∂w=∑i=1N(σ(zi)−yi)x(i) \frac{\partial \mathcal{L}}{\partial \boldsymbol{w}} = \sum_{i=1}^N \bigl(\sigma(z_i) - y_i\bigr)\boldsymbol{x}^{(i)} ∂w∂L​=i=1∑N​(σ(zi​)−yi​)x(i)

可以看到，梯度直接等于“预测概率减真实标签”再乘输入。这里 σ(z)\sigma(z)σ(z) 是模型给出的预测概率，yyy 是离散标签 000 或 111。二者相差多大，梯度就有多大。

如果改用均方误差 12(y−σ(z))2\frac{1}{2}\bigl(y - \sigma(z)\bigr)^221​(y−σ(z))2，其梯度会变成 (σ(z)−y)σ′(z)\bigl(\sigma(z) - y\bigr)\sigma'(z)(σ(z)−y)σ′(z)。多出来的 σ′(z)\sigma'(z)σ′(z) 在输出概率接近 000 或 111 时趋近于 000，即便预测错得很厉害，梯度也可能变得很小，参数更新近乎停滞。交叉熵损失则把梯度直接绑在预测误差上，训练通常更快也更稳定。

Softmax 回归可以看作 Logistic 回归在多分类场景下的推广。当类别数 K>2K > 2K>2 时，标签转化为独热向量 y∈{0,1}K\boldsymbol{y} \in \{0, 1\}^Ky∈{0,1}K。我们假设标签服从分类分布，也就是一次试验对应 KKK 个互斥结果。为每个类别 kkk 分配独立的权重 wk\boldsymbol{w}_kwk​ 和偏置 bkb_kbk​，得到分数 zk=wk⊤x+bkz_k = \boldsymbol{w}_k^\top \boldsymbol{x} + b_kzk​=wk⊤​x+bk​，再通过 softmax 转成概率：

P(y=k∣x)=ezk∑j=1Kezj P(y = k \mid \boldsymbol{x}) = \frac{e^{z_k}}{\sum_{j=1}^K e^{z_j}} P(y=k∣x)=∑j=1K​ezj​ezk​​

这保证了输出向量为正且和为 1。

单个样本的似然就是其真实类别 yyy 对应的预测概率：

p(y∣x)=∏k=1Ky^k1{y=k}=y^y p(y \mid \boldsymbol{x}) = \prod_{k=1}^K \hat{y}_k^{\mathbf{1}\{y = k\}} = \hat{y}_y p(y∣x)=k=1∏K​y^​k1{y=k}​=y^​y​

负对数似然给出多类交叉熵损失：

L=−∑i=1Nlog⁡y^y(i)=−∑i=1N∑k=1Kyk(i)log⁡y^k(i) \mathcal{L} = -\sum_{i=1}^N \log \hat{y}_{y^{(i)}} = -\sum_{i=1}^N \sum_{k=1}^K y_k^{(i)} \log \hat{y}_k^{(i)} L=−i=1∑N​logy^​y(i)​=−i=1∑N​k=1∑K​yk(i)​logy^​k(i)​

其中 yk(i)y_k^{(i)}yk(i)​ 是样本 iii 的独热标签在第 kkk 维上的取值。

同样地，对单个样本，损失对分数 zkz_kzk​ 的偏导为：

∂L∂zk=y^k−yk \frac{\partial \mathcal{L}}{\partial z_k} = \hat{y}_k - y_k ∂zk​∂L​=y^​k​−yk​

它依然保持“预测概率减真实标签”的形式。

更一般地，在机器学习和深度学习中，我们常常已知一个真实分布 PPP，希望模型拟合出一个尽可能接近它的分布 QQQ。最常见的做法就是最小化二者的交叉熵。Logistic 回归、Softmax 回归，以及使用 softmax 输出层的神经网络，本质上都落在这条统一的线上。
