连接主义的包容：从 Logistic 到 Softmax 回归

对于一个离散随机变量 XXX，其概率分布为 PPP，香农熵定义为：
[
H(P) = -\sum_{k} p(k) \log p(k)
]

熵是混乱程度的度量，熵上升意味着纯度下降。

考虑一个真实分布为 PPP，我们用分布 QQQ 来编码事件，定义交叉熵为：
[
H(P, Q) = -\sum_{k} p(k) \log q(k)
]

如何度量Q和P的相似程度，即**如何根据训练样本选择最好的模型参数？**这里我们引入最大似然估计。

假设我们拥有训练集 D={(x(i),y(i))}i=1N\mathcal{D} = \{(\boldsymbol{x}^{(i)}, y^{(i)})\}_{i=1}^ND={(x(i),y(i))}i=1N​，包含 NNN 个独立同分布的样本。我们希望模型在给定输入 x\boldsymbol{x}x 时输出对标签 yyy 的预测概率 p(y∣x;θ)p(y \mid \boldsymbol{x}; \boldsymbol{\theta})p(y∣x;θ)，其中 θ\boldsymbol{\theta}θ 代表如权重 w\boldsymbol{w}w 和偏置 bbb一样的参数。

一个合理的标准是：选择参数 θ\boldsymbol{\theta}θ，使得模型观测到这些真实标签的概率最大。 这个概率就是似然函数：
[
L(\boldsymbol{\theta}) = \prod_{i=1}^N p(y^{(i)} \mid \boldsymbol{x}^{(i)}; \boldsymbol{\theta})
]
对其取负对数，我们的目标是最小化负对数似然，即将负对数似然作为损失函数：
[
\mathcal{L}(\boldsymbol{\theta}) = -\ell(\boldsymbol{\theta}) = -\sum_{i=1}^N \log p(y^{(i)} \mid \boldsymbol{x}^{(i)}; \boldsymbol{\theta})
]
训练目标就是最小化 L(θ)\mathcal{L}(\boldsymbol{\theta})L(θ)。

又有，对于分类标签，真实分布可以看作一个极端概率分布，即One-Hot 向量，真实类概率为 1，其余为 0。则有：
[
-\log p(y^{(i)} \mid \boldsymbol{x}^{(i)}) = -\sum_k \mathbf{1}{y^{(i)}=k} \log p(k \mid \boldsymbol{x}^{(i)}) = H(P_i, Q_i)
]
这正是每个样本的真实分布 PiP_iPi​ 与模型预测分布 QiQ_iQi​ 的交叉熵。所以，最大化似然，等价于最小化交叉熵损失。

Logistic 回归

对于二分类，标签 y∈{0,1}y \in \{0, 1\}y∈{0,1}。我们假设它服从伯努利分布，其唯一参数为 P(y=1)P(y=1)P(y=1)。Logistic 回归让线性组合经过 sigmoid 函数来输出这个概率：
[
z = \boldsymbol{w}^T \boldsymbol{x} + b, \qquad \hat{y} = \sigma(z) = \frac{1}{1+e^{-z}}
]
于是：
[
P(y=1 \mid \boldsymbol{x}) = \hat{y}, \qquad P(y=0 \mid \boldsymbol{x}) = 1-\hat{y}
]
统一写为：
[
p(y \mid \boldsymbol{x}) = \hat{y}^y (1-\hat{y})^{1-y}
]

整个训练集的似然函数为：
[
L = \prod_{i=1}^N \hat{y}_i^{y_i} (1-\hat{y}i)^{1-y_i}
]
取对数，得到对数似然：
[
\ell = \sum{i=1}^N \big[ y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}i) \big]
]
最大化 ℓ\ellℓ 等价于最小化其负值，即二元交叉熵损失：
[
\mathcal{L} = -\sum{i=1}^N \big[ y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i) \big]
]

为什么它能避免梯度消失？

先考虑单样本损失：
[
\mathcal{L} = -\big[ y\log\sigma(z) + (1-y)\log(1-\sigma(z)) \big]
]
求 ∂L∂w\frac{\partial \mathcal{L}}{\partial \boldsymbol{w}}∂w∂L​，由链式法则：
[
\frac{\partial \mathcal{L}}{\partial \boldsymbol{w}} = \frac{\partial \mathcal{L}}{\partial z} \cdot \frac{\partial z}{\partial \boldsymbol{w}} = \frac{\partial \mathcal{L}}{\partial z} \cdot \boldsymbol{x}
]
先求 ∂L∂z\frac{\partial \mathcal{L}}{\partial z}∂z∂L​（标量对标量）：
[
\begin{aligned}
\frac{\partial \mathcal{L}}{\partial z}
&= -y \cdot \frac{1}{\sigma(z)}\sigma'(z) - (1-y) \cdot \frac{1}{1-\sigma(z)}(-\sigma'(z)) \
&= -y\frac{\sigma'(z)}{\sigma(z)} + (1-y)\frac{\sigma'(z)}{1-\sigma(z)}
\end{aligned}
]
代入 sigmoid 的导数 σ′(z)=σ(z)(1−σ(z))\sigma'(z) = \sigma(z)(1-\sigma(z))σ′(z)=σ(z)(1−σ(z))：
[
\begin{aligned}
\frac{\partial \mathcal{L}}{\partial z}
&= -y\frac{\sigma(z)(1-\sigma(z))}{\sigma(z)} + (1-y)\frac{\sigma(z)(1-\sigma(z))}{1-\sigma(z)} \
&= -y(1-\sigma(z)) + (1-y)\sigma(z) \
&= -y + y\sigma(z) + \sigma(z) - y\sigma(z) \
&= \sigma(z) - y
\end{aligned}
]
因此：
[
\frac{\partial \mathcal{L}}{\partial \boldsymbol{w}} = (\sigma(z) - y) \cdot \boldsymbol{x}
]
对于 NNN 个样本，梯度就是求和（或平均）：
[
\frac{\partial \mathcal{L}}{\partial \boldsymbol{w}} = \sum_{i=1}^N (\sigma(z_i) - y_i) \boldsymbol{x}^{(i)}
]

观察可知，其梯度直接等于“预测概率 − 真实标签”乘以输入。 这里 σ(z)\sigma(z)σ(z) 就是模型输出的预测概率值（在 0 到 1 之间），yyy 是离散标签 0 或 1。二者直接相减，误差多大，梯度就多大。

又有，倘若我们使用均方误差 12(y−σ(z))2\frac{1}{2}(y-\sigma(z))^221​(y−σ(z))2，其梯度为 (σ(z)−y)⋅σ′(z)(\sigma(z)-y) \cdot \sigma'(z)(σ(z)−y)⋅σ′(z)。多出来的 σ′(z)\sigma'(z)σ′(z) 在输出概率接近 0 或 1时趋近于 0，导致即便预测错得离谱，梯度也极小，参数更新近乎停滞。交叉熵损失让梯度直接正比于预测误差，训练既快又稳定。正是这一特性，让交叉熵成为分类任务的不二之选。

而Softmax回归作为Logistic回归的泛化，考虑多分类任务。当类别数 K>2K > 2K>2 时，标签转化为独热向量 y∈{0,1}K\boldsymbol{y} \in \{0,1\}^Ky∈{0,1}K（只有一个位置为 1）。我们假设标签服从分类分布（一次试验，K 个互斥结果）。为每个类别 kkk 分配独立的权重 wk\boldsymbol{w}_kwk​ 和偏置 bkb_kbk​，算得分数 zk=wkTx+bkz_k = \boldsymbol{w}_k^T \boldsymbol{x} + b_kzk​=wkT​x+bk​。再通过 softmax 函数转化为概率：
[
P(y=k \mid \boldsymbol{x}) = \frac{e^{z_k}}{\sum_{j=1}^K e^{z_j}}
]
这保证了输出向量为正且和为 1。

单个样本的似然即其真实类别 yyy 对应的预测概率：
[
p(y \mid \boldsymbol{x}) = \prod_{k=1}^K \hat{y}k^{\mathbf{1}{y=k}} = \hat{y}y
]
负对数似然给出多类交叉熵损失：
[
\mathcal{L} = -\sum{i=1}^N \log \hat{y}{y^{(i)}} = -\sum_{i=1}^N \sum_{k=1}^K y_k^{(i)} \log \hat{y}_k^{(i)}
]
其中 yk(i)y_k^{(i)}yk(i)​ 是样本 iii 独热标签的第 kkk 维。

同样地，对单个样本，损失对分数 zkz_kzk​ 的偏导为：
[
\frac{\partial \mathcal{L}}{\partial z_k} = \hat{y}_k - y_k
]
依然是 “预测概率 − 真实标签” 形式。

更一般地，在机器学习和深度学习中通常已知一个概率分布P，算法要拟合一个概率分布Q，使得后者尽可能接近前者。可以通过最小化二者的交叉熵实现。logistic回归、softmax回归以及使用softmax输出层的人工神经网络都是典型代表。
